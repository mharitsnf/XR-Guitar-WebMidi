B-Bender XR Guitar

How to use:

  1. Enable IAC Driver Bus in MIDI Studio
  2. Connect the guitar MIDI input to this device
  3. Select the MIDI input and output
  4. Start the server
  5. Enter the shown IP address into the Oculus application
  6. Setup the bending area in Oculus
  7. Start playing!
